 Built on a hybrid Mamba-transformer architecture, these models dramatically reduce memory usage—by over 70%—and can run on lower-cost GPUs, making them ideal for long-context tasks and multi-session workloads. Released under the permissive Apache 2.0 license, Granite 4.0 includes several variants like H-Small, H-Tiny, and H-Micro, tailored for different deployment needs. Notably, Granite 4.0 is the first open-source model family to achieve ISO/IEC 42001 certification for AI governance, and all model checkpoints are cryptographically signed to ensure authenticity. Available on platforms like IBM watsonx.ai, Hugging Face, Kaggle, and Docker Hub, Granite 4.0 sets a new standard for transparency, scalability, and cost-effectiveness in enterprise AI
